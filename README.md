
# Disability in Children's Books Website
Website created by **V. Go** using open-source **Charadex** framework

>  > [!IMPORTANT]
>  **Created with a [Charadex v1.5.0 Google Sheet](https://docs.google.com/spreadsheets/d/1GwgfLizD3HQCieGia6di-TfU4E3EipT9Jb0BDZQwNak/edit?gid=2014536518#gid=2014536518) and a modified version of [Charadex Version 1.5.5](https://charadex.com/versions.html?profile=v1.5.5).**


## Current public features
- List books in a "masterlist" that highlights its traits, such as type(s) of disability featured, perspective, culture, inclusion of pride, author, and illustrator.
- Sort through books in the masterlist with a versatile filtration system with several parameters.
- List "celebrated" books to highlight reccomended readings.
