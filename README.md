
# Book website mockup

>  > [!IMPORTANT]

*Please click on this link to see the Google sheets being used as the backend: [Book site google sheets](https://docs.google.com/spreadsheets/d/17nCCHXHEUcD5gHTTKCE_rZqTt0V_i3VOs8uVo-aOK3A/edit?gid=1149579922#gid=1149579922)*

## Current features
- List books on a "masterlist" that highlights its different traits, such as type of disability, perspective, culture, sexual/gender orientation inclusion, etc etc
- Sort through books in the masterlist with a versatile filtration system
- Make profiles for authors, and have books authored by them appear automatically in their profile
- Edit log system to keep reference of any edits made in the Google Sheets
- Terminology Page to help clarify how the project classifies certain terminology for the books
- Shows most recent entries added to the masterlist.

## Potential Features that can be added (if interested in continuing to use Charadex)
